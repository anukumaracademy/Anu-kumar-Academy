# Anu Kumar Academy - Project Overview

Anu Kumar Academy is a premium Hindi tuition platform that provides personalized 1-1 coaching for students and professionals. This modern website serves as the academy's official digital presence, optimized for trust, authority, and high-conversion lead generation.

## 🚀 Technology Stack

The website is built using a modern, high-performance tech stack:
*   **React + Vite**: For a lightning-fast user experience and optimized production building.
*   **TypeScript**: Ensures the code is reliable, professional, and easy to maintain.
*   **Tailwind CSS**: Used for the entire styling system, allowing for a sleek, responsive, and mobile-first design.
*   **Framer Motion**: Powering all the "Premium" animations (scroll reveal, hover effects, and page transitions).
*   **Lucide React**: Provides the clean, modern icons used throughout the site.
*   **Vercel**: Handles the high-performance hosting and automatic Global CDN deployment.

## ✨ Key Features & Functionality

*   **Interactive Navigation**: A glassmorphism-inspired sticky navbar that tracks the user's location.
*   **Responsive Course Catalog**: Detailed breakdowns of Academics, Spoken Hindi, and Sabha courses with specific "Learning Outcomes" and "Target Audience" sections.
*   **Flexible Schedule Display**: Clear presentation of class slots with options for custom timing requests.
*   **Dynamic Social Proof**: A dedicated testimonials section with star ratings and community feedback to build trust.
*   **Global Enquiry System**: Direct access to academy support from any page via the global "Enquire Now" triggers.

## 📞 How Enquiries Work (End-to-End)

The site features a seamless lead generation flow:

1.  **User Action**: A student or parent clicks the **"Enquire Now"** button or the floating **WhatsApp** icon.
2.  **Enquiry Form**: A professional popup modal appears, displaying an embedded **Google Form**.
3.  **Submission**:
    *   Once the form is filled out, the data is transmitted directly to the **Anu Kumar Academy Google Account**.
    *   **Data Protection**: No third-party servers store this data; it goes straight from the user to the client.
4.  **Storage**:
    *   The responses are automatically logged in the **"Responses"** tab of the Google Form.
    *   The client can access a synchronized **Google Sheet** (Excel-style) to manage and track all leads in one place.
5.  **Response Protocol**: The site promises a response within **24 hours**, setting a professional expectation for the client.

## 🛠️ Maintenance & Deployment

*   **Dual-GitHub Update**: The project is synchronized across two separate GitHub repositories (Developer and Client) simultaneously to ensure redundancy and accessibility.
*   **Vercel Edge Hosting**: The site is globally deployed. Every update pushed to GitHub is automatically built and made live in under 60 seconds.
*   **SPA Routing Fix**: Implemented `vercel.json` configuration to ensure the site remains fully functional even when a user refreshes the page on deep sub-paths (like /courses).
